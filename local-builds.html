Local builds

Create builds locally using development versions of elecord-web.

  1. Download a webapp.zip from a recent GitHub workflow run
    github.com/elecordapp/elecord-web/actions
  2. Extract elecord-vb3a70e6-js-a29b873.tar.gz to ./v1.2.3/
  3. Rename tarball to also match elecord-v1.2.3.tar.gz
  4. Edit PACKAGE_URL_PREFIX in ./scripts/fetch-package.ts
  5. Start a live preview server from this file in vscode
    open-vsx.org/extension/ms-vscode/live-server
